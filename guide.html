🎯 你需要匯出 3 種 CSV 檔案

#檔案類型提供什麼分析頻率
1️⃣每日互動 CSV每日粉專互動趨勢、負面意見監控每月
2️⃣貼文明細 CSV ⭐每則貼文成效、TOP 10、時段分析每月
3️⃣熱門內容格式 CSV各格式（相片/影片/連結）表現比較每月
✅ 匯出後：把 3 個檔案交給 Kristin，或自行拖到分析後台上傳。

🌐 開始前準備

  • 用 電腦瀏覽器 開啟（手機介面沒有匯出功能）
  • 你的帳號需要 粉專管理員權限
  • 建議用 Chrome 或 Safari
統一入口：business.facebook.com
登入後，左上角切換到你的粉專（例如「XX 店 Apple Shop」）。

1 每日互動 CSV

📍 路徑：洞察報告 → 成果 → 匯出
  1. 左側選單點 「📊 洞察報告」
  2. 點左側子選單 「📈 成果」
  3. 右上角日期範圍改成 過去 6 個月（或自訂 2026/1/1 ~ 今天）
  4. 找到 「內容互動總數」 卡片
  5. 卡片右上角點 「⬇️ 匯出 ▾」 → 選 CSV
  6. 下載完成 ✅
📄 檔名範例：Jan-01-2026_Jul-10-2026_xxxxxxx.csv
💡 每個卡片都可以匯出：瀏覽次數、瀏覽者、內容互動總數、連結點擊次數。這裡最重要的是「內容互動總數」。

2 貼文明細 CSV ⭐

📍 路徑：內容 → 貼文 → 匯出
  1. 左側選單點 「📝 內容」（在洞察報告下方）
  2. 點頂部分頁 「貼文」
  3. 右上角日期範圍改成 過去 6 個月
  4. 右上角點 「⋯ 更多」 或 「⬇️ 匯出」
  5. 選 CSV 格式
  6. 下載完成 ✅
📄 檔名範例：Jan-01-2026_Jun-30-2026_xxxxxxx.csv
⭐ 這份是最重要的檔案：能分析每則貼文的觀看、觸及、互動、分享等所有數據，做出 TOP 10 排行。

3 熱門內容格式 CSV

📍 路徑：洞察報告 → 內容 → 熱門內容格式 → 匯出
  1. 左側選單點 「📊 洞察報告」
  2. 點左側子選單 「📋 內容」（不是總覽的內容）
  3. 往下捲，找到 「熱門內容格式」 卡片
  4. 卡片右上角點 「⋯」 或 「⬇️ 匯出」 → CSV
  5. 下載完成 ✅
📄 檔名範例：熱門內容格式.csv

📊 打開分析後台

🔀 進階：合併模式（每月增量更新）

什麼時候用？

  • 首次上傳：不用勾（覆蓋模式即可）
  • 後續每月：勾 🔀 合併模式，只匯出當月新資料就好

操作範例

📅 1 月首次：匯出 2026/1/1 ~ 2026/1/31 → 不勾合併 → 上傳
📅 2 月更新：匯出 2026/2/1 ~ 2026/2/28 → 勾合併 ✅ → 上傳
📅 3 月更新：匯出 2026/3/1 ~ 2026/3/31 → 勾合併 ✅ → 上傳

👉 資料會累積起來，不會被覆蓋

❓ 常見問題

Q1：找不到匯出按鈕？

Meta 介面常常改版，如果找不到：
• 每個資料卡片右上角都會有 ⬇️ 或 ⋯
• 或整個頁面右上角
• 截圖問 Kristin，會幫你指路

Q2：匯出檔案打不開？

用 Excel 或 Numbers 打開。
如果亂碼：另存新檔 → 編碼選 UTF-8

Q3：日期範圍有限制嗎？

Meta 一般最多讓你匯出過去 2 年。
建議每月匯出一次，累積資料。

Q4：資料多久更新？

貼文數據會持續更新 24-72 小時。
建議下個月月初才匯出上個月的資料（比較準）。

Q5：需要付費嗎？

❌ 完全免費，只要有粉專管理員權限就能匯出。

Q6：我的資料會被其他店看到嗎？

不會 🔒 每家店的資料獨立儲存在自己的瀏覽器，不會共享。
只有 Kristin 統一收集時才會看到全部。

Q7：換電腦後資料還在嗎？

資料存在瀏覽器本機，換電腦要重新上傳 CSV。
建議定期用後台的「📤 匯出備份」功能存 JSON 檔。
